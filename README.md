# 📊 Project Overview #
## This project uses data visualization tools to uncover long-term climate patterns and warming trends across different regions in Africa. It highlights regional baseline variations, seasonal volatility, and shifting temperature distributions over four decades. ##

🔑 Key Insights
Regional Variances: Clear baseline temperature gaps demonstrating Cameroon’s tropical climate compared to Tunisia’s Mediterranean/desert climate.

Shifting Distributions: Analysis of Senegal's historical weather data shows a flattening distribution curve, with a significant increase in the frequency of higher temperature days (86°F–95°F) in the last two decades (2000–2023) compared to the 1980–2000 baseline.

Global Warming Trajectory: A shared, parallel upward trajectory across the analyzed nations, visualizing steady climate shifts over time.
